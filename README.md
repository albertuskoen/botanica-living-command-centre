# Botanica Living Group — Command Centre v1.3

**Premium Executive Operating System**

## Quick Start

```bash
npm install
npm run dev        # http://localhost:5173
npm run build      # production build → dist/
```

## Deploy to Vercel

Push folder contents to GitHub root → import in Vercel → auto-detects Vite → deploy.

## Install on Samsung / Android (PWA)

1. Open in Chrome or Samsung Internet
2. Menu (⋮) → "Add to Home Screen"
3. Opens in standalone mode — no browser bar
